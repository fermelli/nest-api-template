# NestJS API Template

## Description

This is a template for a NestJS API. It is based on the [Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
